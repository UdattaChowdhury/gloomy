# What is this?

Beautiful glooms shortcuts

# Installations

`npm i gloomy --save`

Then...

```
import { gloomy } from 'gloomy';

gloomy({
    shadow_type: 'hard',
    padding: true,
})
```

# Options

gloomy supports 2 options, both of which are optional;

* *shadow_type* - _soft / hard_ (Defaults to hard)
* *padding* - _boolean_ (Defaults to true)
